# Genetic Wellness Agent 

This project is a Genetic Wellness Agent built with Google ADK (Agent Development Kit).  
Unlike FAQ-based agents, this one leverages the foundational Large Language Model (LLM) directly, allowing it to answer general questions about genetic wellness without a predefined dataset.   

---

## üìå Features
- Built using Google ADK with the `gemini-2.0-flash` model.  
- Acts as a Genetic Wellness Assistant.  
- Provides clear, polite answers to customer queries.
 - Declines unrelated queries with:  
  "I specialize only in genetic wellness and cannot answer that."
- Encourages consulting a healthcare professional for medical concerns.    
- Developed with assistance from ChatGPT.
